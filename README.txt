Step 1: Create Python Environment.

Step 2: Install all requirements from requirements.txt file in python Environment.

Step 3: Run qr_code_reader.py file.

Step 4: Choose 1 for Scan code from image.
NOTE: Image is hardcoded. you can change image from code.

Step 5: Choose 2 for Scan code from Webcam.
NOTE: QR Code should be valid.
